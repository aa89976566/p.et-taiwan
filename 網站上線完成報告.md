## 📊 資料表說明

### 為什麼資料表是空的？

**這是完全正常的！** 就像新開的商店：

```
目前狀態 = 新店開幕

✅ 店面裝修好 = 資料庫建立
✅ 貨架擺好了 = 資料表建立
⏳ 等待進貨   = 需要添加商品
⏳ 等待顧客   = 需要用戶註冊
```

### 資料如何填充？

```
用戶註冊 → users 資料表有資料
添加商品 → products 資料表有資料
用戶下單 → orders, order_items 資料表有資料
使用優惠券 → coupon_usage 資料表有資料
```

**你不需要手動操作資料庫！** 

使用網站介面，資料會自動儲存！
