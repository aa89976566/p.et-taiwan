# 🔄 清除瀏覽器緩存 - 查看 Logo 更新

## 問題
在 localhost:8080 看不到 logo 更新，這是因為瀏覽器緩存了舊版本。

## 解決方案

### 方法 1：硬刷新（最簡單）
**Mac:**
- `Cmd + Shift + R` 
- 或 `Cmd + Option + R`

**Windows:**
- `Ctrl + Shift + R`
- 或 `Ctrl + F5`

### 方法 2：清除瀏覽器緩存

**Chrome/Edge:**
1. 按 `F12` 打開開發者工具
2. 右鍵點擊瀏覽器的刷新按鈕
3. 選擇「清空快取並強制重新整理」

**Firefox:**
1. 按 `Ctrl + Shift + Delete` (Windows) 或 `Cmd + Shift + Delete` (Mac)
2. 選擇「快取」
3. 點擊「立即清除」

**Safari:**
1. 按 `Cmd + Option + E` 清除緩存
2. 然後按 `Cmd + R` 刷新

### 方法 3：無痕模式測試
1. 打開無痕/隱私瀏覽模式
2. 訪問 `http://localhost:8080/index.html`
3. 這樣可以繞過緩存

### 方法 4：檢查文件是否正確
在終端執行：
```bash
cd /Users/ming/Documents/GitHub/p.et-taiwan
grep -n "svg width=\"48\"" index.html
```

應該看到第 71 行有 SVG logo。

### 方法 5：直接查看源代碼
1. 在瀏覽器中按 `Cmd + Option + U` (Mac) 或 `Ctrl + U` (Windows)
2. 查看源代碼
3. 搜索 "svg width" 應該能找到新的 logo

## 預期結果

更新後應該看到：
- ✅ 藍色圓形背景的 logo（不再是爪子圖標）
- ✅ 黑白相間的狗頭
- ✅ 粉紅色舌頭向上伸出

## 如果還是看不到

1. **檢查控制台錯誤**：
   - 按 `F12` 打開開發者工具
   - 查看 Console 標籤是否有錯誤

2. **檢查 Network 請求**：
   - 在開發者工具的 Network 標籤
   - 刷新頁面
   - 查看 `index.html` 的狀態碼（應該是 200）

3. **確認服務器正在運行**：
   ```bash
   # 確認服務器在 8080 端口運行
   lsof -i :8080
   ```

4. **重啟服務器**：
   ```bash
   # 停止當前服務器
   # 然後重新啟動
   cd /Users/ming/Documents/GitHub/p.et-taiwan
   python3 -m http.server 8080
   ```
