# 🐕 匠寵電商 - 完整設計方案
## 從狗狗視角看世界的寵物電商平台

---

## 🎨 設計概念

### 核心理念
**"從狗狗的眼睛看世界"** - 低角度、溫暖、充滿好奇心的視覺體驗

### 視覺特色
- 🐕 **低視角構圖**：商品圖採用低角度拍攝，模擬狗狗仰視的視角
- 🐾 **可愛元素**：狗掌印、骨頭、球球等裝飾元素
- 🌤️ **溫暖色調**：藍色（天空）+ 橘色（溫暖陽光）
- 📸 **動態效果**：hover 時商品微微彈跳，模擬狗狗興奮的樣子

---

## 🎨 配色系統

### 主色調
```css
--sky-blue: #4A90E2         /* 天空藍 - 主按鈕、標題 */
--warm-orange: #FF8C42      /* 溫暖橘 - 次要按鈕、強調 */
--grass-green: #7BC043      /* 草地綠 - 成功提示 */
--soft-white: #FAFBFC       /* 柔和白 - 背景 */
--paw-brown: #8B6F47        /* 狗掌棕 - 點綴色 */
```

### 使用場景
| 顏色 | 使用場景 |
|-----|---------|
| 藍色 | 主按鈕、導航、標題、連結 |
| 橘色 | 次要按鈕、促銷標籤、特價標示 |
| 綠色 | 成功訊息、庫存充足標示 |
| 白色 | 背景、卡片 |
| 棕色 | 裝飾、圖示 |

## 📐 頁面架構

### 1. 商品列表頁 (products.html)
**設計重點：**
- 網格布局（3列，響應式）
- 每個商品卡片採用低角度圖片
- Hover 時卡片微微上浮 + 陰影加深
- 快速購買按鈕（橘色，顯眼）
- 左側篩選欄（分類、價格、品牌）

**狗狗視角元素：**
```
┌─────────────────────────────┐
│     🐕 從地面仰望的視角      │
│                              │
│  ┌───┐  ┌───┐  ┌───┐       │
│  │ 📦│  │ 📦│  │ 📦│       │
│  │商品│  │商品│  │商品│       │
│  └───┘  └───┘  └───┘       │
│   🐾     🐾     🐾         │
└─────────────────────────────┘
```

---

### 2. 商品詳情頁 (product-detail.html)
**設計重點：**
- 大圖展示（60% 寬度）+ 詳細資訊（40% 寬度）
- 圖片輪播（可點擊切換）
- 規格選擇器（尺寸、口味）
- 數量選擇器（大號按鈕）
- 立即購買按鈕（藍色，固定在右下角）

**互動效果：**
- 圖片放大功能
- 滾動時購買按鈕固定顯示
- 加入購物車後彈出提示（狗掌印動畫）

---

### 3. 購物車頁 (cart.html)
**設計重點：**
- 商品列表（圖片 + 名稱 + 規格 + 數量 + 小計）
- 優惠券輸入框
- 運費計算
- 總金額顯示（大字，醒目）
- 結帳按鈕（橘色，大號）

**狗狗元素：**
- 空購物車時顯示可愛的狗狗插圖 🐕
- "主人，購物車空空的，幫我買點零食吧！"

---

### 4. 結帳頁面 (checkout.html)
**設計重點：**
- 步驟指示器（1.確認商品 → 2.填寫資料 → 3.選擇付款 → 4.完成訂單）
- 收件資訊表單
- 地址選擇器
- 付款方式選擇
- 訂單總覽（右側固定）

**簡化流程：**
- 一頁完成結帳（不要分多頁）
- 自動填入會員資料
- 記住上次使用的地址

---

### 5. 會員中心 (member-dashboard.html)
**設計重點：**
- 左側選單（個人資料、訂單記錄、地址管理、收藏商品）
- 右側主內容區
- 最近訂單快速查看
- 積分/優惠券顯示

**狗狗元素：**
- 個人頭像處顯示狗狗大頭貼
- "你的毛孩子會喜歡這些商品"推薦區

---

### 6. 個人資料頁 (member-profile.html)
**表單欄位：**
- 大頭貼上傳（圓形）
- 姓名、Email、電話
- 寵物資訊（名字、品種、年齡）
- 偏好設定（通知方式）

---

### 7. 訂單記錄頁 (member-orders.html)
**功能：**
- 訂單列表（日期、訂單號、金額、狀態）
- 狀態篩選器（全部、待付款、已付款、已出貨、已完成）
- 訂單詳情展開
- 再次購買按鈕

---

### 8. 地址管理頁 (member-addresses.html)
**功能：**
- 地址卡片列表
- 新增/編輯/刪除地址
- 設定預設地址
- 標籤（家、公司）

---

### 9. 收藏商品頁 (member-favorites.html)
**功能：**
- 網格展示收藏的商品
- 快速加入購物車
- 移除收藏
- 降價通知設定

---

```css
.bone-divider {
    background-image: url('data:image/svg+xml,<svg>...</svg>');
    height: 20px;
    background-repeat: repeat-x;
}
```

### 狗狗跳躍動畫
```css
@keyframes dogJump {
    0%, 100% { transform: translateY(0); }
    50% { transform: translateY(-10px); }
}
```

## 📱 響應式設計

### 斷點
- 手機：< 640px（1列）
- 平板：640px - 1024px（2列）
- 桌機：> 1024px（3-4列）

### 導航
- 桌機：水平導航
- 手機：漢堡選單 + 抽屜式側欄

## 🎯 核心使用者體驗

### 首次訪問流程
1. 看到可愛的首頁（狗狗視角的溫馨場景）
2. 瀏覽特色商品（低角度拍攝，吸引眼球）
3. 點擊商品查看詳情
4. 加入購物車
5. 快速結帳（簡化表單）
6. 完成購買（成功頁面顯示可愛的狗狗慶祝動畫）

### 回訪用戶流程
1. 自動登入（記住我功能）
2. 看到個人化推薦
3. 查看最近訂單
4. 再次購買最愛的商品
