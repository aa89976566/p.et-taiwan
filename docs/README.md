# 文檔目錄

本目錄包含項目的所有文檔。

## 📁 目錄結構

- `guides/` - 重要使用指南和說明文檔
- `archive/` - 歷史文檔和完成報告（僅供參考）

## 📖 重要文檔

請查看 `guides/` 目錄下的指南文檔：

- **首頁展示邏輯與順序調整指南.md** - 如何調整首頁區塊順序和顯示
- **首頁產品展示設定指南.md** - 如何設定首頁產品展示
- **伺服器啟動完整指南.md** - 後端服務器啟動說明
- **線上付款使用指南.md** - 綠界金流使用說明

## 📚 歸檔文檔

`archive/` 目錄包含所有歷史開發文檔、完成報告和修復記錄。這些文檔僅供參考，不會被 Git 追蹤。
