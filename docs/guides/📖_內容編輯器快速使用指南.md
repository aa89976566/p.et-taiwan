## ⚡ 快速開始

### 1. 開啟編輯器

**兩種方式**：

#### 方式 A：從後台儀表板
1. 開啟 `admin-login.html` 
2. 登入後台

#### 方式 B：直接開啟

## 📝 5 分鐘快速教學

### 任務 1：修改首頁標題（1 分鐘）

```
1. 點擊「首頁橫幅」
2. 在「主標題」輸入框輸入新標題
3. 右側立即顯示預覽
4. 點擊頂部「儲存變更」
✅ 完成！
```

### 任務 2：更換首頁圖片（1 分鐘）

```
1. 點擊「首頁橫幅」
2. 找到「背景圖片 URL」
3. 點擊「上傳圖片」按鈕
4. 選擇您的圖片
5. 查看預覽
6. 點擊「儲存變更」
✅ 完成！
```

### 任務 3：設定按鈕連結（1 分鐘）

```
1. 點擊「首頁橫幅」
2. 找到「按鈕設定」區塊
3. 主要按鈕：
   - 按鈕文字：開始測驗
   - 跳轉連結：quiz-nutrition.html
4. 次要按鈕：
   - 按鈕文字：查看產品
   - 跳轉連結：#products
5. 點擊「儲存變更」
✅ 完成！
```

### 任務 4：調整物流設定（1 分鐘）

```
1. 點擊「物流選項」
2. 調整宅配到府運費：80 → 100
3. 調整 7-11 運費：60 → 50
4. 勾選「啟用免運優惠」
5. 設定免運門檻：1000 元
6. 點擊「儲存變更」
✅ 完成！
```

### 任務 5：發布到網站（1 分鐘）

```
1. 確認所有變更都已完成
2. 點擊「預覽變更」查看效果
3. 點擊「發布到網站」
4. 確認發布
✅ 完成！內容已上線！
```

## 💡 常見問題

### Q1：儲存後內容會消失嗎？
**A**：不會！內容會自動儲存到 localStorage，關閉瀏覽器後再打開仍會保留。

### Q2：如何查看變更效果？
**A**：
1. **即時預覽**：右側預覽區立即顯示
2. **全頁面預覽**：點擊「預覽變更」在新視窗查看

### Q3：「發布到網站」是什麼意思？
**A**：將儲存的內容套用到前端網站，讓訪客看到最新內容。

### Q4：可以回復到之前的版本嗎？
**A**：目前版本會保留最後一次儲存的內容。建議重要變更前先記錄原始內容。

### Q5：按鈕連結無效怎麼辦？
**A**：
- 檢查連結是否正確
- 頁面連結需包含 `.html`
- 錨點連結需以 `#` 開頭
- 外部連結需包含 `https://`

### Q6：物流設定會立即生效嗎？
**A**：需要點擊「發布到網站」才會套用到前端結帳頁面。

## 🛠️ 技巧與訣竅

### 技巧 1：使用錨點連結
快速跳到頁面特定區塊：
```
#products    → 跳到產品展示
#quiz        → 跳到智能測驗
#subscription → 跳到訂閱方案
#footer      → 跳到頁尾
```

### 技巧 2：批次編輯
同時編輯多個區塊，最後一起儲存：
```
1. 編輯首頁橫幅
2. 切換到物流選項
3. 編輯物流設定
4. 切換到頁尾資訊
5. 編輯聯絡資訊
6. 統一點擊「儲存變更」
```

### 技巧 3：預覽再發布
確保內容正確後再發布：
```
1. 完成所有編輯
2. 點擊「預覽變更」
3. 在新視窗仔細檢查
4. 確認無誤後點擊「發布到網站」
```

### 技巧 4：備份重要內容
重大變更前記錄原始內容：
```
1. 截圖原始設定
2. 或將原始文字複製到記事本
3. 進行變更
4. 如需回復，手動輸入原始內容
```
