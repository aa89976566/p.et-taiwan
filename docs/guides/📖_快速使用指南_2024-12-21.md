## 🎨 內容編輯器

### 快速開始 (3 步驟)

#### 步驟 1: 開啟編輯器
```
方式 A: admin-dashboard.html → 系統設定 → 內容編輯器
方式 B: 直接開啟 admin-content-editor.html
```

#### 步驟 2: 編輯內容

**首頁橫幅**
- **標題**: 網站主標題（大標）
- **副標題**: 說明文字（小標）
- **主要按鈕**: 
  - 按鈕文字: 例如「探索產品」
  - 連結: 例如 `#products` 或 `products.html`
  - 新分頁: ☑️ 勾選後在新分頁開啟
- **次要按鈕**:
  - 按鈕文字: 例如「開始測驗」
  - 連結: 例如 `quiz.html`
  - 新分頁: ☑️ 勾選後在新分頁開啟
- **背景圖片**: 輸入圖片 URL

**頁尾資訊**
- 公司名稱
- 聯絡電話
- Email
- 地址
- 版權聲明

#### 步驟 3: 儲存與預覽

| 按鈕 | 功能 | 效果 |
|------|------|------|
| 💾 **儲存** | 儲存所有修改 | 立即同步到前端頁面 |
| 👁️ **預覽** | 查看實際效果 | 開啟 index.html 新分頁 |

---

### 📝 實用範例

#### 範例 1: 修改首頁標題和按鈕

**情境**: 想要更改首頁的標題和按鈕文字

1. 開啟內容編輯器
2. 在「首頁橫幅」區塊修改：
   ```
   標題: 給毛孩最好的照顧
   副標題: 手作零食 × 健康生活
   主要按鈕文字: 立即選購
   主要按鈕連結: #products
   ```
3. 點擊「預覽」查看效果
4. 點擊「儲存」保存修改
5. 開啟 index.html 查看成果

#### 範例 2: 設定按鈕在新分頁開啟

**情境**: 希望「開始測驗」按鈕在新分頁開啟

1. 找到「次要按鈕」設定
2. 勾選「在新分頁開啟」☑️
3. 儲存
4. 測試: 點擊按鈕會在新分頁開啟 quiz.html

#### 範例 3: 更換背景圖片

**情境**: 想要更換首頁橫幅的背景圖

1. 準備圖片 URL（例如從 Unsplash）
2. 在「背景圖片 URL」欄位貼上：
   ```
   https://images.unsplash.com/photo-1548199973-03cce0bbc87b
   ```
3. 點擊「預覽」立即查看效果
4. 滿意後點擊「儲存」

## 🚚 物流設定

### 快速開始 (3 步驟)

#### 步驟 1: 開啟物流設定
```
方式 A: admin-dashboard.html → 物流管理 → 物流設定
方式 B: 直接開啟 admin-shipping.html
```

#### 步驟 2: 設定物流選項

**宅配到府**
- ☑️ 啟用/停用
- 運費: NT$ ___
- 配送時間: ___ 個工作天
- 物流公司: 
  ```
  黑貓宅急便
  新竹貨運
  嘉里大榮
  宅配通
  ```
  _(每行一個)_

**7-11 超商取貨**
- ☑️ 啟用/停用
- 運費: NT$ ___
- 配送時間: ___ 個工作天

**全家超商取貨**
- ☑️ 啟用/停用
- 運費: NT$ ___
- 配送時間: ___ 個工作天

**免運門檻**
- 金額: NT$ ___

#### 步驟 3: 測試

| 按鈕 | 功能 |
|------|------|
| 💾 **儲存設定** | 儲存物流設定 |
| 🧪 **測試結帳** | 開啟結帳頁面測試 |

---

### 📝 實用範例

#### 範例 1: 設定基本物流選項

**情境**: 提供宅配和 7-11 取貨，宅配 NT$ 80，7-11 NT$ 60

1. **宅配到府**:
   - ☑️ 啟用
   - 運費: `80`
   - 配送時間: `3-5`
   - 物流公司:
     ```
     黑貓宅急便
     新竹貨運
     ```

2. **7-11 超商取貨**:
   - ☑️ 啟用
   - 運費: `60`
   - 配送時間: `2-3`

3. **全家超商取貨**:
   - ☐ 停用 _(不勾選)_

4. 點擊「儲存設定」
5. 點擊「測試結帳」查看效果

#### 範例 2: 設定免運門檻

**情境**: 滿 NT$ 1000 免運費

1. 在「免運門檻」輸入: `1000`
2. 儲存設定
3. 測試: 購物車金額 ≥ NT$ 1000 時，運費顯示 NT$ 0

#### 範例 3: 臨時停用某個物流方式

**情境**: 7-11 系統維護，臨時停用

1. 找到「7-11 超商取貨」
2. 取消勾選「啟用」☐
3. 儲存
4. 結帳頁面將不再顯示 7-11 選項
5. 恢復時重新勾選即可

## ❓ 常見問題

### Q1: 修改後沒有看到變化？

**解決方法**:
1. **確認已儲存**: 點擊「儲存」按鈕
2. **清除快取**: 按 `Ctrl + Shift + R` (Windows) 或 `Cmd + Shift + R` (Mac)
3. **重新載入**: 關閉並重新開啟頁面

### Q2: 預覽按鈕沒反應？

**可能原因**:
- 瀏覽器攔截了彈出視窗

**解決方法**:
1. 允許瀏覽器彈出視窗
2. 或直接開啟 index.html 查看效果

### Q3: 如何檢查設定是否已儲存？

**檢查方法**:
1. 按 `F12` 開啟開發者工具
2. 點擊「Application」或「應用程式」分頁
3. 左側選擇「Local Storage」
4. 查看：
   - `websiteContent` - 網站內容設定
   - `shippingSettings` - 物流設定

### Q4: 物流選項在結帳頁面沒顯示？

**檢查清單**:
- [ ] 是否已勾選「啟用」？
- [ ] 是否已點擊「儲存設定」？
- [ ] 是否已在結帳頁面重新整理？
- [ ] 是否有 JavaScript 錯誤？(按 F12 查看 Console)

### Q5: 如何恢復預設設定？

**方法 1 - 清除 localStorage**:
```javascript
// 在瀏覽器 Console 執行
localStorage.removeItem('websiteContent');
localStorage.removeItem('shippingSettings');
location.reload();
```

**方法 2 - 手動重新設定**:
直接在編輯器中重新輸入預設值

### Q6: 可以同時編輯嗎？

**回答**: 
- ✅ 可以在多個分頁編輯
- ⚠️ 最後儲存的會覆蓋之前的
- 💡 建議: 一次只在一個分頁編輯

### Q7: 按鈕連結可以填什麼？

**常見連結類型**:

| 類型 | 範例 | 說明 |
|------|------|------|
| 錨點 | `#products` | 跳轉到同頁面的區塊 |
| 內部頁面 | `quiz.html` | 開啟網站內的其他頁面 |
| 外部網站 | `https://example.com` | 開啟外部網站 |
| 郵件 | `mailto:hello@example.com` | 開啟郵件程式 |
| 電話 | `tel:+886-800-123-456` | 撥打電話（手機） |

### Q8: 圖片 URL 從哪裡來？

**取得圖片 URL 的方法**:
1. **Unsplash**: https://unsplash.com (免費高品質圖片)
2. **自己上傳**: 上傳到圖床服務後複製 URL
3. **現有圖片**: 右鍵點擊圖片 → 複製圖片位址

**範例 URL**:
```
https://images.unsplash.com/photo-1548199973-03cce0bbc87b
```

## 🎓 進階技巧

### 技巧 1: 批量測試連結

修改多個連結後，可以依序測試：
1. 儲存設定
2. 開啟 index.html
3. 逐一點擊按鈕測試連結

### 技巧 2: 使用相對路徑

內部頁面使用相對路徑更靈活：
```
✅ 推薦: quiz.html
❌ 不推薦: https://yourdomain.com/quiz.html
```

### 技巧 3: 測試不同裝置

1. 按 `F12` 開啟開發者工具
2. 點擊「裝置模擬」圖示
3. 選擇不同裝置查看效果
