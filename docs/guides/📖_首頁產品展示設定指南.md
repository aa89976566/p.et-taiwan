## 🔧 自訂首頁產品顯示

### 方法 1：透過後台首頁管理（推薦）

1. 開啟 `admin-homepage.html`
2. 找到「**產品系列**」區塊
3. 點擊「**編輯**」按鈕
4. 在「**產品選擇**」區塊中，點擊「**選擇產品**」
5. 選擇要顯示的產品
6. 設定顯示數量（預設 3 個）
7. 儲存並發布

### 方法 2：直接修改產品資料

在後台產品管理中：
- 修改產品的**銷量**（salesCount），銷量越高的產品會優先顯示
- 確保產品**狀態**為「上架中」
- 確保產品**分類**為 `toys`

---

## ⚠️ 重要提醒

### 產品 ID 必須完全一致

如果產品 ID 與預設不符，首頁可能無法正確顯示難度標籤：
- `toy-snuffle-mat` → 顯示「初級」綠色標籤
- `toy-puzzle-slider` → 顯示「中級」黃色標籤
- `toy-memory-buttons` → 顯示「高級」紅色標籤

### 產品必須存在於資料庫

如果資料庫中沒有這些產品，首頁會顯示：
> "請先在後台新增玩具產品"

### 自動載入機制

首頁產品是**自動載入**的：
- ✅ 無需手動設定首頁產品
- ✅ 會自動從後端 API 取得最新產品資料
- ✅ 按照銷量排序，顯示最受歡迎的產品

## 📞 遇到問題？

### 問題 1：首頁看不到產品

**解決方法**：
1. 檢查後台是否有新增這些產品
2. 確認產品分類為 `toys`
3. 確認產品狀態為「上架中」
4. 檢查瀏覽器 Console 是否有錯誤訊息
5. 確認後端 API 是否正常運行

### 問題 2：產品顯示順序不對

**解決方法**：
1. 在後台產品管理中，調整產品的「銷量」數字
2. 銷量越高的產品會優先顯示

### 問題 3：想要顯示其他產品

**解決方法**：
1. 在後台新增新產品，分類設為 `toys`
2. 將新產品的銷量設為較高數字
3. 首頁會自動顯示銷量最高的前 3 個玩具產品
