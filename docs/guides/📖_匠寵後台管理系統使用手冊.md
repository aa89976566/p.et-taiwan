## 🚀 快速開始

### 系統需求
- ✅ 現代瀏覽器 (Chrome, Firefox, Safari, Edge)
- ✅ 網路連線
- ✅ 管理員帳號

### 首次使用
1. 開啟瀏覽器
2. 前往後台網址：`https://admin.jiangchong.com`
3. 使用管理員帳號登入
4. 完成！開始管理您的網站

## 🔐 登入系統

### 登入步驟

1. **開啟登入頁面**
   - 網址：`admin-login.html`
   - 或直接訪問：`https://admin.jiangchong.com`

2. **輸入帳號密碼**
   ```
   電子郵件：您的管理員 Email
   密碼：您的密碼
   ```

3. **選擇「記住我」（選用）**
   - 勾選此選項可保持登入狀態 7 天

4. **點擊「登入」按鈕**
   - 登入成功後自動導向儀表板

### 測試帳號

開發/測試環境可使用以下帳號：

| 角色 | Email | 密碼 | 權限 |
|-----|-------|------|------|
| 超級管理員 | admin@jiangchong.com | admin123 | 完全權限 |
| 一般管理員 | manager@jiangchong.com | manager123 | 日常營運 |
| 客服人員 | cs@jiangchong.com | cs123 | 客服相關 |

⚠️ **安全提示**：正式環境請務必更改預設密碼！

### 忘記密碼

1. 點擊登入頁面的「忘記密碼？」
2. 輸入註冊的 Email
3. 系統會寄送重設密碼連結
4. 點擊連結設定新密碼

## 🎯 功能說明

### 用戶管理

#### 查看用戶列表
1. 點擊側邊欄「用戶列表」
2. 顯示所有註冊用戶
3. 支援搜尋、篩選、分頁

#### 用戶詳情
1. 點擊用戶名稱或「查看」按鈕
2. 顯示完整資料：
   - 個人資料
   - 寵物列表
   - 訂單歷史
   - 測驗記錄

#### 編輯用戶
1. 進入用戶詳情頁
2. 點擊「編輯」按鈕
3. 修改資料
4. 點擊「儲存」

#### 停用/啟用用戶
1. 用戶列表中找到目標用戶
2. 點擊「停用」/「啟用」按鈕
3. 確認操作

---

### 產品管理

#### 新增產品
1. 點擊「產品列表」
2. 點擊「新增產品」按鈕
3. 填寫產品資訊：
   - 產品名稱
   - 分類
   - 價格
   - 庫存
   - 描述
   - 圖片
4. 點擊「儲存」

#### 編輯產品
1. 產品列表中找到產品
2. 點擊「編輯」
3. 修改資訊
4. 儲存變更

#### 上傳圖片
1. 編輯產品頁面
2. 點擊「上傳圖片」
3. 選擇檔案（支援 JPG, PNG）
4. 最多 5 張圖片

#### 庫存管理
1. 產品列表顯示當前庫存
2. 點擊庫存數字可快速修改
3. 庫存不足會紅色警示

---

### 訂單管理

#### 查看訂單
1. 點擊「訂單管理」
2. 顯示所有訂單
3. 篩選條件：
   - 訂單狀態
   - 日期範圍
   - 用戶名稱

#### 訂單狀態

| 狀態 | 說明 | 操作 |
|-----|------|------|
| 🟡 待確認 | 新訂單 | 確認 / 取消 |
| 🔵 處理中 | 備貨中 | 標記出貨 |
| 🟣 已出貨 | 運送中 | 更新物流 |
| 🟢 已完成 | 交易完成 | 無 |
| 🔴 已取消 | 訂單取消 | 無 |

#### 處理訂單流程
1. 開啟訂單詳情
2. 確認商品資訊
3. 更新訂單狀態
4. 填寫物流資訊（如適用）
5. 系統自動發送通知Email

#### 列印出貨單
1. 訂單詳情頁
2. 點擊「列印出貨單」
3. 瀏覽器列印功能

#### 退款處理
1. 訂單詳情頁
2. 點擊「退款」按鈕
3. 選擇退款原因
4. 輸入退款金額
5. 確認退款

---

### 測驗管理

#### 查看測驗列表
1. 點擊「測驗列表」
2. 顯示所有測驗記錄
3. 篩選：

   - 平均分數

#### 匯出報表
1. 測驗統計頁面

## ❓ 常見問題

### Q1: 忘記密碼怎麼辦？
**A**: 點擊登入頁面的「忘記密碼？」，輸入您的 Email，系統會寄送重設密碼連結。

### Q2: 如何修改個人資料？
**A**: 點擊右上角用戶選單 → 個人設定 → 修改資料 → 儲存。

### Q3: 訂單無法更新狀態？
**A**: 請檢查：
1. 您的角色權限
2. 訂單當前狀態（某些狀態無法變更）
3. 網路連線是否正常

### Q4: 圖片上傳失敗？
**A**: 確認：
1. 檔案格式（僅支援 JPG, PNG）
2. 檔案大小（不超過 5 MB）
3. 網路連線

### Q5: 數據不準確？
**A**: 可能原因：
1. 資料尚未同步（等待幾分鐘）
2. 篩選條件設定錯誤
3. 清除瀏覽器快取後重試

### Q6: 系統速度很慢？
**A**: 建議：
1. 使用現代瀏覽器
2. 清除瀏覽器快取
3. 檢查網路連線
4. 避開高峰時段

### Q7: 如何匯出資料？
**A**: 
1. 進入相關模組（如訂單、測驗）
2. 點擊「匯出」按鈕
3. 選擇格式（Excel / CSV）
4. 下載檔案

### Q8: 誰可以存取後台？
**A**: 僅限獲得授權的管理員，包括：
- 超級管理員
- 一般管理員
- 客服人員
- 倉庫人員

### Q9: 如何申請管理員帳號？
**A**: 請聯絡超級管理員，由其在「管理員管理」中新增您的帳號。

### Q10: 系統支援手機使用嗎？
**A**: 是的！系統採用響應式設計，支援手機、平板、桌機使用。
